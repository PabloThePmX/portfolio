Lógica Informática

2020 - Now

During my day-to-day work, I contribute to various projects that support the ERP system:

  • CAD Plugin: I develop new functionalities, fix bugs, and refactor some routines in a plugin that exports CAD tool structures to the ERP.
    - Developed using C# with WPF, Entity Framework, and PostgreSQL.

  • Reporting Software: I restructured the software used for viewing and assembling ERP reports. Currently, I handle all activities related to this tool. One of my main tasks was to adapt the software to run on a separate thread and exclusively in 64-bit, significantly improving performance when dealing with large images.
    - Developed using C# and WPF.

  • AI Chat: I contributed to the final stages of implementing a generative AI-based chatbot that integrates information from Lógica’s softwares. I was also responsible for transforming this application into an API, enabling its integration with Ez Chat.
    - Developed using GeneXus, WorkWithPlus, and PostgreSQL.

  • ERP DLLs: I develop and maintain various DLLs in C# for use within the company's ERP system.
